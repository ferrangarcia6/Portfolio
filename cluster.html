Cluster de raspberry pi

He montado un cluster de raspberrys pi con kubernetes en concreto k8s. Este cluster consta de un maestro y tres esclavos. Corre varios contenedores con servicios como por ejemplo:

  • Nginx tanto como servicio web como proxy
  • Nodejs
  • Servidor multimedia que detallo aqui con automatizador de torrents con Tranmission y flexget
  • Servidor vpn
  • Owncloud
  • Servidor Samba
  • Servicio broker mqtt como mosquitto
  • Node-red para simplificar las automatizaciones sobre dispositivos IoT
  • Network Server LoRaWan
  • Pyhton
  • Servidor de Bakcups
  • Home Assitant
