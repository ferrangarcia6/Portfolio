SOBRE MI:

Hola, me llamo Ferran Garcia y desde bien pequeño la tecnologia ya levantó el interes en mi.

A día de hoy he desarrollado muchos proyecto tanto personales como profesionales en los que he aprendido y me he formado para adquirir un conocimiento muy amplio en muchos ambitos de la tecnologia. Pero sin duda lo que más me motiva de este campo es la continua evolución de poder seguir aprendiendo.

Tecnologias

Desarrollo Front-End

Desarrollo de aplicaciones principalmente con HTML, CSS y Javascript y Frameworks como Angular y React.


Desarrollo Back-end y Otros

Desarrollo en la parte del backend, creación y administración de bases de datos y API Rest. He tranajado con tecnologias como Javascript con NodeJs, php, java servlets, MySQL, MongoDB

IoT

Experiencia en tecnologias IoT, como LoRaWan, Sigfox y Narrowband. Conocimineto de protocolos de comunicación como Coap, Http o Mqtt. Gestión de Network Server, Aplication server y creación de proyectos con módulos y sensores como esp8266, esp32, MKR nb1500 y mucho más.

Administración de Sistemas

Administración de sistemas informaticos basados en linux. Gestión de redes. Gestión de routers, firewals. Implantación de microservisiós. Conocimiento del shell de linux. Automaticación de tareas con scrips ps1 bat o con python. Tecnologias como Kubernetes, docker.
