Desarrollo Web

Desarrollo de una aplicación web creada con Html, Css y Javascript en el Front-end y con Php en el Back-end junto una base de datos Mariadb. Esta aplicación esta orientada a la gestión de despacho de abogados, pero luego fué adapatad para otros usos.

Algunas de las funcionalidades de esta aplicación son:
  • Un CRUD de clientes e información necesaria, junto la implementación de un buscador con filtos
  • Un dashboard con información estadística de los clientes así como graficos de contabilidad
  • La asociación de número de procedimiento asignados por el Juzgado
  • La automatización de documentos como facturas, minutas o citaciones, tanto en PDF para imprimir como por correo electronico
  • Gestión de documentación de cada cliente organizada y automatizada para que se pueda trabajar con discos en red con SMB o directamente desde la aplicación
  • Un calendario sincronizado con las cuentas de Google calendar de cada usuario para la organización de citas y reuniones
